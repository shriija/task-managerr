# Kanvas Project Overview

Kanvas is a collaborative task management application for teams and individuals who want a clear place to organize work, track progress, and keep everyone aligned.

## What The App Is For

The app helps people plan work in a visual way. Users can create boards for projects, break work into lists, and manage individual tasks as cards. It is designed to make it easy to see what is being worked on, what is still pending, and what has already been completed.

## Main Areas Of The App

### Home Page

The home page introduces Kanvas and explains the main benefits of the app. It gives new users a clear path to sign up and returning users a way to log in.

### Dashboard

The dashboard is the starting point after signing in. It brings together the user’s boards in one place and lets them move between:

- their own boards
- boards shared with them by other people
- deleted items that can be recovered or removed permanently

From here, users can also create a new board.

### Board Workspace

The board workspace is where most of the daily work happens. Each board can contain multiple lists, and each list can contain multiple cards. This gives users a simple visual flow for moving work from one stage to another.

Users can:

- create and rename lists
- add new tasks as cards
- open tasks to update their details
- move tasks between lists
- search through tasks on a board

## Ways People Can Work In Kanvas

### Board View

This is the main task board. It is laid out like a workflow, so users can keep tasks grouped by stage and update them as work moves forward.

### My Tasks View

This view helps users focus on the work assigned to them. It acts like a personal task list inside a shared project.

### Calendar View

The calendar view shows tasks by due date. It helps users understand upcoming deadlines, spot overdue work, and plan their time more easily.

### Trash View

The trash view gives users a safety net. Deleted boards, lists, and cards can be restored if they were removed by mistake, or deleted forever if they are no longer needed.

## Collaboration Features

Kanvas is built for teamwork. Board owners can invite other people to join a board, either directly or through a shareable invite link. Once invited, collaborators can work on the same board and stay in sync with the rest of the team.

Boards also show their members, making it easy to see who is involved in a project.

## Task Management Features

Tasks in Kanvas can include useful details such as:

- a title
- a description
- a priority level
- a due date
- a status such as to do, in progress, or completed
- labels for quick visual organization

This makes it easier to keep tasks clear, organized, and easy to follow.

## User Experience Summary

In practice, Kanvas gives a user a simple path:

1. Sign in or create an account.
2. Open the dashboard.
3. Create or open a board.
4. Add lists and tasks.
5. Update tasks as work progresses.
6. View deadlines in the calendar.
7. Invite teammates and collaborate on the same work.
8. Recover items from trash if needed.

## Overall Value

Kanvas brings project planning, task tracking, collaboration, and deadline visibility into one place. It is meant to help teams stay organized without making the experience feel heavy or complicated.
